# AfterAll

Mobile horror game — run-based gameplay with augment selection. *(Working title — in active development)*

## 🎮 About

Short horror runs where each playthrough can branch into different builds through mid-run augment picks. Stealth, tension, and mystery. Core design is evolving during development.

**Status:** v0.1 Foundation — movement + prototype environment + interaction system

## 🛠️ Built With

- Unity 6 (3D URP)
- C#
- Platform: Mobile (iOS + Android) — keyboard prototype for now

## 📁 Project Structure

```
Assets/_AfterAll/
  Scenes/       — Prototype_Room and future scenes
  Scripts/
    Player/     — Movement, camera
    Interaction/ — IInteractable, PlayerInteractor
    Core/       — GameManager (run state later)
  Prefabs/
  Materials/Greybox/
```

## 🕹️ Controls *(prototype)*

- WASD — Move
- Mouse — Look
- E — Interact
